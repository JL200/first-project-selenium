This is my first python code created for studies using Selenium Library

This Project consistis in use Selenium for acess "https://ge.globo.com/", the most popular site about sports in Brazil to get the classification of Brasileirão Serie A, the first Division of Brasilian Football league.

The code gets the Brasileirão table in two diferente parts:

1. The club ranking list.

2. Data on the teams in the league such as: Number of wins, draws, defeats and other data.
